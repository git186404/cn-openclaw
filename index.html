项目概览

Moltbot 是一个可自托管的个人 AI 助手，主打“本地优先、始终在线”。它以 Gateway 控制平面为核心，统一管理会话、渠道、工具与事件，并可通过 CLI 和客户端应用随时调用。

你可以通过引导向导（moltbot onboard）完成安装，适配 macOS、Linux、Windows (WSL2)。同时支持 npm、pnpm 或 bun 安装。

推荐运行方式

moltbot onboard --install-daemon

控制平面

ws://127.0.0.1:18789 作为 Gateway 接入点

订阅模型

支持 Anthropic / OpenAI OAuth 与 API Key 轮换

核心亮点

从多渠道收件箱到语音唤醒，全面覆盖个人助手所需能力。

多渠道收件箱

支持 WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、Matrix、Zalo 等渠道，还可接入 WebChat 与蓝牙扩展。

本地 Gateway 控制平面

单一 WebSocket 网络管理会话、事件、配置、工具与自动化，让你的助手稳定运行在本地设备上。

语音唤醒与 Talk Mode

支持 macOS、iOS、Android 的持续语音对话，结合 ElevenLabs 实现常驻语音助手体验。

工具与 Canvas

内置浏览器控制、Canvas A2UI 视觉工作区、节点能力（相机、录屏、位置、通知）以及 Cron 自动化。

多 Agent 路由

按工作区或群组隔离会话，支持群聊 mention 激活、回复路由和 chunk 发送策略。

伴随应用

提供 macOS 菜单栏应用、iOS/Android 节点模式，随时调用设备端能力。

安全与运维

默认 DM 配对策略、模型故障切换、Tailscale/SSH 远程接入与完整诊断工具。

安装与快速开始

Node 22+ 运行时，推荐使用 CLI 向导完成安装。

推荐安装

npm install -g moltbot@latest
# 或: pnpm add -g moltbot@latest

moltbot onboard --install-daemon

向导会安装 Gateway 守护进程，保证后台运行。

快速启动

moltbot gateway --port 18789 --verbose

moltbot message send --to +1234567890 \
  --message "Hello from Moltbot"

moltbot agent --message "Ship checklist" \
  --thinking high

可选择将回复发送回任意连接渠道。

安全默认设置

Moltbot 默认将未知私信视为不可信输入，并通过配对码流程进行授权。你也可以为群组或频道设置 allowlist 与 mention 激活策略。

  • • DM 配对策略：dmPolicy="pairing"，未知发送方需先验证。
  • • 公开 DM 需显式开启：dmPolicy="open" + allowFrom。
  • • 使用 moltbot doctor 检查配置风险。

远程访问

支持 Tailscale Serve/Funnel 或 SSH 隧道，让 Gateway 保持回环绑定也可安全远程访问。

沙箱模式

可设置 agents.defaults.sandbox.mode: "non-main" 让非主会话在 Docker 沙箱中运行。
